Para que serve?

Se você precisa alternar o sistema InovaFarma entre diferentes bases de dados (matriz, filiais, ambiente de testes, backup, etc.), o procedimento padrão envolve abrir o arquivo de configuração Conecta, editar a string de conexão manualmente e salvar — toda vez que precisa trocar.

O InovaConecta automatiza esse processo: você seleciona a instância SQL Server, escolhe o banco desejado em uma lista, e o aplicativo gera e substitui o arquivo Conecta correto. Sem digitar string, sem risco de erro de sintaxe.

Funcionalidades

Detecção automática

Identifica instâncias SQL Server locais instantaneamente e busca instâncias da rede em segundo plano, sem travar a abertura do app.

Troca em um clique

Selecione o banco e o arquivo Conecta é regenerado e substituído automaticamente. Pronto pra abrir o InovaFarma já apontando pra base correta.

Títulos personalizados

Dê nomes amigáveis aos seus bancos (ex: "Matriz", "Filial Centro", "Teste") e adicione observações para identificar rapidamente cada ambiente.

Inicia com o Windows

Configure para abrir automaticamente junto com o sistema operacional e tenha o InovaConecta sempre à mão na barra de tarefas.

Iniciar InovaFarma

Após trocar o banco, abre o InovaFarma direto pelo próprio aplicativo. Fluxo completo em poucos cliques.

Filtragem inteligente

Bancos de sistema (master, msdb, tempdb) e bases internas do InovaFarma são automaticamente ocultados — só aparece o que importa.

Como usar

  1. 1

    Baixe e instale

    Baixe o InovaConecta_Setup.exe da página de releases e execute o instalador. O atalho aparece na Área de Trabalho e no menu Iniciar.

  2. 2

    Selecione a instância SQL

    Escolha uma instância detectada automaticamente ou digite o nome (ex: SERVIDOR\SQL2016) e informe usuário e senha do SQL Server.

  3. 3

    Busque os bancos

    Clique em Buscar Bancos e o app lista todas as bases disponíveis na instância selecionada.

  4. 4

    Troque o Conecta

    Selecione o banco desejado e clique em Trocar Conecta. Pronto — o InovaFarma já vai abrir apontando pra base nova.
